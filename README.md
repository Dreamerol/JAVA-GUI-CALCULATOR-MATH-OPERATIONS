## 🚀 Overview

This project is a **simple calculator application** built using Java with a **Graphical User Interface (GUI)**. It allows users to perform **basic arithmetic operations** through an interactive visual interface.

## ✨ Features

- User-friendly graphical interface  
- Buttons for digits **0–9**  
- Basic arithmetic operations:  
  - Addition (+)  
  - Subtraction (−)  
  - Multiplication (×)  
  - Division (÷)  
- **Clear (C)** button to reset input  
- Display screen for input and results

## 🖥️ How It Works

1. The user clicks buttons to input numbers and select operations.  
2. The calculator processes the input internally.  
3. The result is displayed on the screen.  
4. All interactions are handled via **GUI events** (button clicks).
